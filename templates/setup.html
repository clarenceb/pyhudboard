Pyhudboard setup

Please enter the name you'd like to show on the dashboard


Step 2:
Drag those links to your browser's bookmark bar
claim  |  clear

Step 3:
Go to a hudson build page (eg: http://dashboard02:9080/job/ad-acceptance/) and click "claim" to put your name on it!
Click 'clear' to clear any name showing on the build. Note: If a build goes green, any associated name will be cleared. If it's blue, the name is still there but not displayed.
